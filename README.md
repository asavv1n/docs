# Тестовая документация

В рамках курса составил тестовую документацию для модулей регистрации, авторизации и каталога [интернет магазина](https://demoshopping.ru/): 
- Составил [Чек-лист](https://docs.google.com/spreadsheets/d/1g_aWjd-ikdDJmyzsuvHONihdAf3bWKsN0Di4unSBWQY/edit?usp=sharing) на основе требований к веб-приложению.

- Практиковался в использовании платформы для управления тестированием [QASE](https://qase.io/). На основе чек-листа сформировал несколько [тест-кейсов](https://github.com/asavv1n/docs/blob/62db262494126ce26fb7399afbeaf6db12edf2a5/%D0%A2%D0%B5%D1%81%D1%82%20%D0%BA%D0%B5%D0%B9%D1%81%D1%8B%20G8-2024-08-23.pdf) для тестирования модулей регистрации, авторизации и каталога.

- С подготовленными тест-кейсами сделал [тестовый прогон](https://github.com/asavv1n/docs/blob/b3e923c173dc45cfc037f8e2a4ac994a3d8ee2dc/G8-Test%2Brun%2B2024_08_24.pdf) (test run).

- По результатам тестового прогона составил [отчёты о дефекте](https://docs.google.com/spreadsheets/d/1xo9n6xjhPbTWVoYlao8wDUa0QuEemWP5/edit?usp=sharing&ouid=114018588172278355978&rtpof=true&sd=true).
